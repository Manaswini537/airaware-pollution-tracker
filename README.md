# AirAware – Air Pollution Exposure Tracker

AirAware is a web application that helps users understand air pollution levels in different cities.  
The system retrieves real-time Air Quality Index (AQI) data and provides pollution exposure insights and health recommendations.

## Features
- Real-time AQI tracking for cities
- AQI category classification (Good, Moderate, Unhealthy, etc.)
- Personal pollution exposure score calculation
- Health recommendations based on AQI levels
- Interactive map visualization
- Pollution impact chart

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Python (Flask)  
Libraries: Chart.js, Leaflet.js  
API: World Air Quality Index (WAQI)
